# Changelog

All notable changes to SYNAPSE will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2025-01-XX

### Added
- **AI CLI Integration System**: Comprehensive support for AI CLI tools
  - Aider integration
  - GPT-Pilot support
  - Cursor CLI support
  - Continue integration
  - Custom tool configuration
  - Auto-detection of installed tools

- **Scripting Engine**: Full scripting support
  - Lua scripting
  - JavaScript execution
  - Python support
  - Shell script execution
  - Script editor with syntax highlighting
  - Script library management

- **Enhanced Theme System**: Professional theming
  - 5 built-in themes (Dark, Light, Synthwave, Dracula, Nord)
  - Custom theme creation
  - Theme import/export
  - Color customization
  - Font customization

- **Accessibility Features**: For hearing impaired and accessibility
  - Text-to-Speech (TTS) support
  - Speech recognition
  - High contrast mode
  - Large text option
  - Screen reader support

- **How-To Guides**: Built-in documentation
  - Getting started guide
  - AI integration guide
  - Scripting guide
  - Theme customization guide
  - Troubleshooting guide

- **Changelog System**: Update notes and version history
  - Version history viewer
  - Detailed change logs
  - Categorized changes (Added, Changed, Fixed, etc.)

- **Code Editor Enhancements**:
  - Syntax highlighting (Rust, Python, JavaScript, and more)
  - Line numbers
  - Code tokenization
  - Multiple language support

### Changed
- Improved UI with tab-based navigation
- Enhanced settings with more customization options
- Better integration between components
- Improved performance and stability

### Fixed
- Terminal rendering issues
- Memory leaks in long-running sessions
- Settings persistence bugs

## [0.1.0] - 2025-01-XX

### Added
- Initial release
- Terminal grid with tiling layouts
- PTY-based terminal integration
- Roster sidebar with file tree
- Lens overlay code editor
- Report window for agent analytics
- Log tracking panel with filtering
- Status bar with system information
- Agent management system
- Settings system with JSON persistence
